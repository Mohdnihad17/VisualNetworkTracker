        ⚠️ CONNECTION LOST

        THE PROBLEM:
        You are opening index.html directly in the browser.
        This means no data loads because the Flask backend is not running.

        THE FIX — DO THIS EVERY TIME:

        1. Open a terminal (Command Prompt or PowerShell on Windows)
        2. Navigate to your project folder: cd path/to/VisualNetworkTracker
        3. Install dependencies (only needed once): pip install flask flask-cors python-dateutil
        4. Run the backend:
          cd backend
          python app.py
        5. You will see:
          "DB initialized."
          "Running on http://0.0.0.0:5000"
        6. NOW open the frontend:
          Open frontend/index.html in your browser
          OR go to http://localhost:5000 if Flask serves the HTML
        7. The green LIVE indicator confirms connection.
        ↺
        Reset Database?
        All detections and traffic data will be cleared. Fresh data will start generating immediately.
